# 天气助手（Weather Assistant）
这是我用 Python 写的第一个项目，可以查询城市实时温度、湿度，并给出穿衣建议。

## 功能
- 查询城市实时温度
- 获取湿度信息
- 根据天气自动生成穿衣建议
- 使用 SQLite 数据库保存查询记录
- 调用高德地图天气 API

## 技术栈
- Python
- SQLite
- 高德地图 API
- VS Code

## 使用方法
1. 输入城市编码（如九江：360400）
2. 程序自动获取天气数据
3. 显示温度、湿度、穿衣建议
4. 数据自动保存到数据库

## 项目特点
- 简单易用
- 数据持久化存储
- 自动生成建议
- 适合初学者学习
